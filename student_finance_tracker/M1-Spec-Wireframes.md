# M1 – Specification & Wireframes

## Project chosen
Student Finance Tracker

## Description
A responsive web application that helps students track expenses, manage budgets, search transactions using regex, and view spending statistics.

## Data Model

javascript
{
  id: "txn_0001",
  description: "Lunch at cafeteria",
  amount: 12.50,
  category: "Food",
  date: "2025-09-29",
  createdAt: "2025-09-29T10:15:30Z",
  updatedAt: "2025-09-29T10:15:30Z"
}

Default Categories:
- Food
- Books
- Transport
- Entertainment
- Fees
- Other

## Accessibility Plan

- Semantic HTML (header, nav, main, section, footer)
- Skip-to-content link
- Labels for all form inputs
- Keyboard navigation support
- Visible focus indicators
- ARIA live regions for status messages
- Sufficient color contrast
